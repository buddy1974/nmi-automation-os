# nmi-automation-os

Centralized automation operating system for NMI — managing workflows, integrations, dashboards, and configuration.

## Project Structure

```
nmi-automation-os/
├── docs/           # Documentation and guides
├── workflows/      # n8n workflow definitions and exports
├── config/         # Environment configs and service settings
├── dashboard/      # Frontend dashboard (framework TBD)
├── database/       # Database schemas, migrations, seeds
├── scripts/        # Utility and automation scripts
├── PROJECT_HEADER.md
├── README.md
├── .env
└── .gitignore
```

## Getting Started

1. Clone the repository
2. Copy `.env` and fill in required values
3. See `docs/` for setup guides
